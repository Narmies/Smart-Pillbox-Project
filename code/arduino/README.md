ჭკვიანი წამლის ყუთი – Arduino-ს კოდი
ეს საქაღალდე შეიცავს "ჭკვიანი წამლის ყუთის" (Smart Pill Box) პროექტის ESP32-ის საბაზისო პროგრამულ უზრუნველყოფას.

ფუნქციები:
რეალური დროის საათზე (RTC) დაფუძნებული განრიგი

განყოფილებების ავტომატური გაღება

სოლენოიდური საკეტის მართვა

წამლის მიღების დადასტურება ლიმიტური გადამრთველების (limit switches) მეშვეობით

ხმოვანი განგაში ბაზერის (buzzer) გამოყენებით

აპარატურა:
ESP32

DS3231 RTC

2x სოლენოიდური საკეტი

2x ლიმიტური გადამრთველი

ბაზერი (Buzzer)

ღილაკი (Push Button)
